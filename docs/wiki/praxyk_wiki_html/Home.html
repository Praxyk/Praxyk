Downloading the Project

git clone https://github.com/jhallard/Praxyk.git
cd Praxyk

Highest Level Directories

  • devops - Contains all code related to development operations.
  • master - Contains all code for the master server.
  • POD - Highest directory for all pre-trained ML-container projects.
  • TLP - High directory for all template ML-container projects.
  • API - Contains all code for the API server.
  • database - Contains all code for the project's databases.
